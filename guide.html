사용 가이드

밴쿠버 한인 업체 목록을 효과적으로 활용하는 방법을 알아보세요

🚀 빠른 시작

🏠

업체 검색하기

홈페이지의 검색바를 사용하여 원하는 업체를 빠르게 찾아보세요.

  • 업체명, 카테고리, 지역으로 검색 가능
  • 실시간 검색 결과 확인
  • 상세 정보 및 위치 확인
🗺️

지도에서 보기

지도 뷰에서 업체들의 위치를 한눈에 확인하고 경로를 찾아보세요.

  • 인터랙티브 지도에서 업체 위치 확인
  • 카테고리별 필터링
  • 내 위치 기반 검색
📝

업체 등록하기

새로운 한인 업체를 등록하여 커뮤니티에 기여해보세요.

  • 간단한 폼 작성으로 등록
  • 업체 정보 상세 입력
  • 검토 후 사이트에 반영

✨ 주요 기능

🔍 검색 기능

검색 팁:

  • • 업체명의 일부만 입력해도 검색됩니다
  • • 카테고리명으로 검색하면 해당 분야 업체들을 찾을 수 있습니다
  • • 지역명(예: 리치몬드, 버나비)으로 검색해보세요

🏷️ 필터 기능

전체 음식점 병원 미용실 기타

필터 팁:

  • • 카테고리 칩을 클릭하여 원하는 업종만 보기
  • • 지역별로 업체를 필터링할 수 있습니다
  • • 정렬 옵션으로 원하는 순서로 정렬하세요

🗺️ 지도 기능

📍
📍
📍

인터랙티브 지도

지도 사용법:

  • • 지도를 드래그하여 원하는 지역으로 이동
  • • 핀을 클릭하면 업체 정보를 확인할 수 있습니다
  • • 내 위치 버튼으로 현재 위치 기준 검색

⭐

추천 업체 활용

홈페이지의 추천 업체 섹션에서 검증된 업체들을 확인해보세요.

🌙

다크모드 사용

우측 상단의 테마 버튼으로 다크모드를 켜고 끌 수 있습니다.

🌐

언어 전환

KO/EN 버튼으로 한국어와 영어를 자유롭게 전환하세요.

📱

모바일 최적화

모든 기능이 모바일에서도 완벽하게 작동합니다.

❓ 자주 묻는 질문

업체 정보가 잘못되었어요. 어떻게 수정하나요?

+

업체 등록 페이지에서 수정 요청을 제출하거나, 각 업체 상세 페이지의 연락처로 직접 문의해주세요.

새로운 업체를 등록하려면 어떻게 하나요?

+

상단 메뉴의 '업체 등록'을 클릭하여 구글폼을 작성해주세요. 검토 후 사이트에 반영됩니다.

지도에서 내 위치가 정확하지 않아요.

+

브라우저의 위치 권한을 허용해주세요. 정확한 위치를 위해 GPS가 켜져있는지 확인해주세요.

업체 정보는 얼마나 자주 업데이트되나요?

+

구글 시트와 연동되어 있어 실시간으로 업데이트됩니다. 변경사항은 즉시 반영됩니다.
